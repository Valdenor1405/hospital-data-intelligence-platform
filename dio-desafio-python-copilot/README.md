\# Desafio DIO - Python com GitHub Copilot



\## 📌 Descrição



Este projeto foi desenvolvido como parte do desafio da DIO: \*\*Utilizando as Ferramentas do GitHub para Solucionar Algoritmos em Python\*\*.



O objetivo é praticar conceitos de Python, Git, GitHub, GitHub Copilot e documentação de projetos.



\## 🚀 Funcionalidade



O programa recebe um número informado pelo usuário e verifica se ele é \*\*par\*\* ou \*\*ímpar\*\*.



\## 🧠 Conceitos praticados



\- Python

\- Funções

\- Condicionais

\- Entrada de dados

\- Git e GitHub

\- Documentação com Markdown
